Instalación de Python

Para comenzar a usar Python, primero debes instalarlo en tu sistema. Puedes descargar la última versión desde el sitio web oficial:

Descargar Python

Sigue las instrucciones de instalación para tu sistema operativo. Asegúrate de agregar Python a las variables de entorno para poder ejecutarlo desde la línea de comandos.

Entorno Virtual

Es recomendable utilizar entornos virtuales para aislar las dependencias de tus proyectos. Puedes crear un entorno virtual con el siguiente comando:

                    
python -m venv mi_entorno

Activa el entorno virtual:

                    
# En Windows
mi_entorno\Scripts\activate

# En macOS y Linux
source mi_entorno/bin/activate

Ejecución de Scripts

Para ejecutar un script de Python, utiliza el siguiente comando:

                    
python mi_script.py

Sintaxis Básica

Python utiliza indentación para definir bloques de código. Aquí tienes un ejemplo:

                    
def saludar(nombre):
    print(f"¡Hola, {nombre}!")

saludar("Mundo")

Tipos de Datos

Python tiene varios tipos de datos integrados:

  • Enteros (int)
  • Flotantes (float)
  • Cadenas (str)
  • Booleanos (bool)
  • Listas (list)
  • Tuplas (tuple)
  • Diccionarios (dict)

Control de Flujo

Python ofrece estructuras de control de flujo como if, for y while:

                    
edad = 20
if edad >= 18:
    print("Eres mayor de edad")
else:
    print("Eres menor de edad")

Funciones

Las funciones son bloques de código reutilizables:

                    
def sumar(a, b):
    return a + b

resultado = sumar(5, 3)
print(resultado)  # Output: 8
